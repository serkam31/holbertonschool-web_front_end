# Holberton School – Web Front End

## Description

Dossier regroupant tous les projets Front-End Web de la formation Holberton School, incluant HTML, CSS, JavaScript et les techniques avancées de développement web.

## Contenu

- `Advanced HTML` – Techniques avancées en structuration et design de pages HTML
- `CSS` – Stylisation et mise en page des pages web
- `JavaScript` – Interactions dynamiques et manipulation du DOM

## Objectifs

- Maîtriser les fondamentaux du développement web
- Construire des pages web modernes et interactives
- Suivre les bonnes pratiques de structuration et de style

## Technologies

- HTML5
- CSS3
- JavaScript

## Exécution

Ouvrir les fichiers `.html` dans un navigateur web pour visualiser les projets.
