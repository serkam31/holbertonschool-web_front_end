# HTML – Advanced

## Description

Projet HTML consacré aux techniques avancées de structuration et de design de pages web.

## Objectifs

- Maîtriser les balises HTML avancées
- Structurer une page web complexe
- Utiliser les formulaires et les tableaux efficacement
- Comprendre l’accessibilité et la sémantique HTML

## Technologies

- HTML5

## Exécution

Ouvrir le fichier `.html` dans un navigateur web.
